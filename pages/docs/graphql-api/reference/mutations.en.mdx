# Mutations

### updateLease 

Use the mutation `updateLease` to apply the tenor, balloon payment and downpayment from `leaseSuggestions` to the lease. 

### requestDeal 

Use the mutation `requestDeal` if you want to finalize the calculation and submit a leasing application to Beequip.
